# Data Use

This repository includes the curated 60-second analysis dataset used to reproduce the manuscript analyses, figures, tables, and supplementary participant traces for:

**From Instantaneous Heart Rate to Long-Horizon Cardiovascular Burden in Naturalistic Daily Life**

The released dataset is:

```text
Data/NUBI_Data_60sec_Level_MASTER_CLEAN.csv
```

The analysis scripts may include interactive options for 10-, 30-, and 60-second resolutions. In the public repository, the 60-second dataset is the manuscript/public-release default. The 10- and 30-second options are intended only for users who have access to matching curated MASTER datasets and matching model-output folders.

## Intended use

The dataset is provided for research transparency, reproducibility, and scholarly reuse. It may be used to reproduce the analyses in the associated manuscript and to support related non-commercial academic research.

## Citation

Users of this dataset or code should cite the associated manuscript:

**From Instantaneous Heart Rate to Long-Horizon Cardiovascular Burden in Naturalistic Daily Life**

A full citation will be added after publication.

## Participant privacy

The dataset was curated for analysis and public release. Users must not attempt to identify, contact, or re-identify individual participants.

Users must not combine this dataset with external datasets for the purpose of participant re-identification.

## Responsible use

Users should not use the dataset for purposes that could harm, profile, discriminate against, or make consequential decisions about individuals or groups.

The dataset is provided for scientific and reproducibility purposes and should not be interpreted as a clinical diagnostic tool or used for medical decision-making.

## Redistribution

If redistributing derived outputs, users should preserve appropriate citation to the associated manuscript and repository.

Redistribution of modified versions of the dataset should clearly indicate that the file has been modified and should not be represented as the original released dataset.

## No warranty

The dataset is provided as-is, without warranty of any kind. The authors make no guarantee that the dataset is suitable for any particular purpose beyond reproducing the analyses described in the associated manuscript.
