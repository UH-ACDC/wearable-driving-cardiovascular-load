# From Instantaneous Heart Rate to Long-Horizon Cardiovascular Burden in Naturalistic Daily Life

This repository contains the analysis code and curated 60-second analysis dataset used to reproduce the figures, tables, modeling results, and supplementary participant traces for the manuscript:

**From Instantaneous Heart Rate to Long-Horizon Cardiovascular Burden in Naturalistic Daily Life**

## Repository scope

This repository starts from the curated analysis-ready 60-second NUBI dataset:

```text
Data/NUBI_Data_60sec_Level_MASTER_CLEAN.csv
```

The repository does **not** include raw sensor-stream preprocessing code or ground-truth dataset-construction code. Instead, the released dataset is treated as the analysis-ready input for the manuscript analyses.

The included scripts reproduce:

- descriptive sample and data-inventory summaries
- baseline-referenced cardiovascular-load analyses
- predictive decomposition of heart rate
- elastic-net modeling results
- long-horizon cardiovascular-burden simulations
- horizon-scaling robustness analyses
- manuscript figures and tables
- supplementary participant-level heart-rate traces

## Data

The analysis-ready dataset is:

```text
Data/NUBI_Data_60sec_Level_MASTER_CLEAN.csv
```

This filename is retained from the original analysis workflow so that the public scripts reproduce the manuscript outputs without path or filename modifications.

The dataset includes the validated 60-second observations used in the manuscript analyses, including participant identifiers, timestamps, study-day labels, heart-rate variables, behavioral context labels, baseline heart rate, driving-related variables, temporal/contextual variables, and participant-level covariates.


## Script map

| Manuscript item | Script | Description |
|---|---|---|
| Main modeling artifacts | `Scripts/00_run_predictive_decomposition.R` | Runs the baseline-only, baseline-plus-tax, and elastic-net predictive decomposition models. Produces out-of-fold predictions, model metrics, feature-importance files, and model artifacts used by later scripts. |
| Figure 1 + Table 1 | `Scripts/01_figure1_table1_exploratory_inventory.R` | Generates exploratory sample summaries, Figure 1, Table 1, and missingness summaries. |
| Figure 2 | `Scripts/02_figure2_representative_trace.R` | Generates the representative participant-level heart-rate trace and supporting subject-level diagnostics. |
| Figure 3 | `Scripts/03_figure3_empirical_cardiovascular_load.R` | Computes baseline-referenced cardiovascular load and generates empirical distribution, time-above-baseline, and variability summaries. |
| Figure 4 | No analysis script | Figure 4 is a conceptual schematic and is not generated by an analysis script. |
| Figure 5 + Table 3 | `Scripts/05_figure5_table3_predictability_decomposition.R` | Generates predictive-decomposition plots and table-style summaries of RMSE, R², changes relative to baseline, and ENet improvement. |
| Figure 6 | `Scripts/06_figure6_enet_modulators.R` | Generates ENet modulator diagnostics, grouped feature importance, signed coefficient summaries, effect curves, and incremental-gain summaries. |
| Figure 7 | `Scripts/07_figure7_long_horizon_simulation.R` | Runs long-horizon cardiovascular-burden simulations and generates annual cumulative NHR-hour summaries. |
| Figure 8 | `Scripts/08_figure8_horizon_scaling_raw_hr.R` | Computes horizon-scaling error metrics and generates the robustness analysis for increasing temporal aggregation windows. |
| Supplementary figure | `Scripts/s01_supplement_activity3_participant_traces.R` | Generates supplementary participant-level heart-rate traces and supporting diagnostics. |


## Recommended run order

Run the scripts from the repository root.

```r
source("Scripts/00_run_predictive_decomposition.R")
source("Scripts/01_figure1_table1_exploratory_inventory.R")
source("Scripts/02_figure2_representative_trace.R")
source("Scripts/03_figure3_empirical_cardiovascular_load.R")
source("Scripts/05_figure5_table3_predictability_decomposition.R")
source("Scripts/06_figure6_enet_modulators.R")
source("Scripts/07_figure7_long_horizon_simulation.R")
source("Scripts/08_figure8_horizon_scaling_raw_hr.R")
source("Scripts/s01_supplement_activity3_participant_traces.R")
```

The first script, `00_run_predictive_decomposition.R`, should be run before Figures 5-8 because it creates the model predictions, model metrics, and feature-importance files used by downstream scripts.

## Outputs

The scripts create `Results/` subfolders containing manuscript figures, tables, model diagnostics, simulation summaries, and intermediate CSV/RDS files.

Typical outputs include:

- manuscript figure PDFs/PNGs
- Table 1 and Table 3 summaries
- missingness summaries
- cross-validated model-performance summaries
- out-of-fold prediction files
- ENet feature-importance summaries
- long-horizon simulation tables
- horizon-scaling robustness tables
- supplementary participant-level trace PDFs

## Reproducibility note

This repository is intended to reproduce the paper-level analyses from the released curated 60-second dataset. It does not reconstruct the dataset from raw wearable, smartphone, vehicle, or ground-truth data streams.
